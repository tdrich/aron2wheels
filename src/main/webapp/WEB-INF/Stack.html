Stacks used in this project

Java

a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. Java was used through out the back end of this project.

MariaDB

a community-developed, commercially supported fork of the MySQL relational database management system. MariaDB was used in the back end of this project to store and recall Ride data

AngularJS

a discontinued free and open-source JavaScript-based web framework for developing single-page applications. It was maintained mainly by Google and a community of individuals and corporations. AngularJS was used in the front end of this project to simplify development

Tomcat

Apache Tomcat is a free and open-source implementation of the Jakarta Servlet, Jakarta Expression Language, and WebSocket technologies. Tomcat allows the front end to communicate with the back end

HTML

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. The lion's share of the front end of this project was created with HTML

CSS

Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML. Along with Bootstrap, many of the elements of this project were styled using CSS

Bootstrap

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. The bootstrap framework was used to streamline CSS for easier development and better page responsiveness
